Build
-----

**Build-time dependencies**

* CMake
* C compiler
* gettext
* ninja(-build) (recommend)
* valac (≥ 0.34)

**Run-time dependencies**

* GLib (≥ 2.38)
* glib-networking
* GTK (≥ 3.22)
* GPGME (For the OpenPGP plugin)
* libgee-0.8 (≥ 0.10)
* libgcrypt (For the OMEMO plugin)
* libsoup (For the HTTP files plugin)
* SQLite3

**Instructions**

    ./configure
    make
    build/dino
